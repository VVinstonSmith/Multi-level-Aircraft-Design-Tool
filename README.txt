本程序用于飞机总体设计中的参数优化。

优化由粗到细, 由整体到局部分为三个阶段（代码放在/Multi-level optimization），分别是：

Level1 optimization:
基于半经验模型的参数优化，
目的是确定飞机总体参数的设计范围。
(通过python实现)

Level2 optimization:
基于简单数值模型+半经验模型的参数优化，
目的是实现飞机的气动，结构综合合计，
数值模型包括：用于计算机翼结构强度, 刚度的梁架有限元，以及用于计算气动力的涡格法。
(通过python实现)

Level3 optimization:
基于复杂数值模型的参数优化，
目的是详细设计机翼的结构厚度分布，并校核机翼的气动弹性特性。
数值模型包括：用于计算机翼受力分布的杆板有限元模型，以及用于计算频域气动力的偶极子格网法。
(MATLAB程序+NASTRAN应用接口 实现)

研究背景概述：
飞机总体设计是一项以力学为主导的系统工程，对型号的研发和使用成本都起着决定性作用。随着计算力学理论的不断发展和应用的日渐成熟，将它们合理高效地整合进总体设计框架成了一个有待解决的任务。
为了解决传统方法对经验的过度依赖以及MDO方法在层次性、计算量方面的不足，本文在飞机总体设计中引入了逐层建模与优化的设计方法，根据已知信息的多少和设计范围的大小将总体设计拆分成了多个阶段，通过明确的分工与继承，基于不同精度的建模与分析方法，使设计对象逐层细化，逐步达到最优的性能。此外，系统化建模和优化设计被引入到其中的每一个阶段，适当的建模方法被选择，加以改进，用来构建以结构重量估算、升阻特性计算和参数化建模为代表的重要环节，从而平衡每一阶段的精度和效率。针对这些内容，本文完成了以下的研究工作：
（1）概念设计阶段采用半经验公式创建飞机的几何、重量、气动模型，通过优化求解初步确定飞机的几何外形和质量质心分布，作为初步设计阶段的参考。
（2）初步设计的第一阶段采用翼盒-梁单元模型模拟飞机结构，用涡格法和准三维方法计算升阻特性，通过气动结构耦合计算和SQP算法对飞机的气动外形、结构布置、结构尺寸进行综合优化设计，获得理想的飞行效率。
（3）初步设计的第二阶段采用杆板有限元对机翼结构进行参数化建模，用偶极子格网法计算频域气动力，引入气动弹性分析环节，以强度、刚度、气动弹性要求为约束对机翼结构厚度的分布规律展开优化设计，以降低结构重量。
